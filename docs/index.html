YEOMIL MONO

Yeomil Mono is a unified CJK-Latin monospace font built by merging the technical precision of Geist Mono with the clarity of Pretendard. Corrects Hangul advance widths to exactly 1200 units (double the Latin 600 units) to guarantee pixel-perfect terminal alignment.

[01]

macOS (Homebrew)

Install system-wide via custom tap cask recipe:

brew tap taevel02/yeomil-mono && brew install --cask font-yeomil-mono
[02]

NPM Package

Add the package dependency to your web project:

npm install yeomil-mono
[03]

CSS Stylesheet

Import built @font-face rules at the top of your global CSS:

@import "yeomil-mono";
